# Multiboard by Espablo
Supla for ESP8266
Firmware v2.7.2

Source code: https://github.com/SUPLA/supla-espressif-esp


Supported modules:
- inCan,<br />
    GPIO2 - Temperature<br />
    GPIO4 - Sensor1<br />
    GPIO16 - Sensor2<br />
    GPIO14 - Button1<br />
    GPIO12 - Button2<br />
    GPIO5 - Relay1<br />
    GPIO13 - Relay2<br />
    GPIO0 - Button Configure (5s)<br />
    GPIO4 - Input CFG (x10)<br />
    GPIO2 - LED CFG<br />

- inCan RollerShutter,<br />
    GPIO2 - Temperature<br />
    GPIO4 - Sensor1<br />
    GPIO16 - Sensor2<br />
    GPIO14 - Button1<br />
    GPIO12 - Button2<br />
    GPIO5 - Relay1<br />
    GPIO13 - Relay2<br />
    GPIO0 - Button Configure (5s)<br />
    GPIO4 - Input CFG (x10)<br />
    GPIO2 - LED CFG<br />

- Sensors x8,
- SONOFF BASIC,
- SONOFF TH,
- SONOFF TOUCH,
