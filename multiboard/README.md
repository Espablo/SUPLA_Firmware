# Multiboard by Espablo
Supla for ESP8266
Firmware v2.7.2

Source code: https://github.com/SUPLA/supla-espressif-esp


Supported modules:
- inCan,
- inCan RollerShutter,
- Sensors x8,
- SONOFF BASIC,
- SONOFF TH,
- SONOFF TOUCH,
- SONOFF TOUCH DUAL,
- SONOFF TOUCH TRIPLE,
- SONOFF_4CH,
- Yunshan,
-------------------------------------------------

Initial parameters for "ESP Falsh Download Tool":

CreystalFreq    26M

SPI SPEED       40 MHz

FLASH MODE      DOUT

BAUDRATE        115200

-------------------------------------------------
